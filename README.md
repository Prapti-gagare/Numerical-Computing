# Numerical Computing

A collection of C++ programs for numerical computing, including matrix operations, root-finding algorithms, and iterative methods.

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

## Project Overview

This repository contains various numerical computing algorithms implemented in C++:
- *Matrix Codes*: Gaussian elimination, LU decomposition, etc.
- **Root-Finding Codes**: Bisection method, Newton-Raphson method, fixed Point method, etc.
- **Iterative Methods Codes**: Jacobi method, Gauss-Seidel method,etc.

---

## Features

- Solve systems of linear equations using different methods
- Find roots of equations numerically
- Perform iterative solutions for large systems
- Modular code structure for easy understanding and reuse
- Input can be provided via files or console (depending on the code)
